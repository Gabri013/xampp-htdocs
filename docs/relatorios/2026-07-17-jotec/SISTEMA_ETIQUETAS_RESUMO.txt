📊 TABELAS CRIADAS
═════════════════════════════════════════════════════════════════════════════

1. etiquetas_impressas (1 tabela)
   ├─ id, os_id, op_numero
   ├─ tipo (qr_os, qr_op, codigo128)
   ├─ conteudo, dados_qr
   ├─ impressoes (contador)
   ├─ usuario_id, data_criacao, data_ultima_impressao
   └─ Índices: os_tipo, op_numero

2. ordens_producao (2 tabelas relacionadas)
   ├─ Tabela Principal:
   │  ├─ id, os_id, numero (OP-001-01)
   │  ├─ status, responsavel_id
   │  ├─ data_inicio, data_termino, prazo_original
   │  ├─ observacoes, criado_em, atualizado_em
   │  └─ Índices: os_numero, status
   │
   ├─ Tabela de Itens:
   │  ├─ id, op_id, os_item_id
   │  ├─ quantidade, quantidade_produzida
   │  ├─ valor_unitario, status
   │  └─ data_conclusao
   │
   └─ Tabela de Etapas:
      ├─ id, op_id, etapa (corte, dobra, etc)
      ├─ status, usuario_id
      ├─ data_inicio, data_conclusao
      ├─ observacao, sequencia
      └─ Índices: op_etapa

🔌 ENDPOINTS REST (API)
═════════════════════════════════════════════════════════════════════════════

POST /api/etiqueta_qrcode.php?acao=gerar_qr_svg
   └─ Parâmetros: os_id
   └─ Retorna: QR-code em URL, etiqueta_id

POST /api/etiqueta_qrcode.php?acao=gerar_qr_svg_op
   └─ Parâmetros: op_numero, os_id
   └─ Retorna: QR-code para O.P.

POST /api/etiqueta_qrcode.php?acao=gerar_codigo128
   └─ Parâmetros: texto, os_id (opcional)
   └─ Retorna: Código de barras 128

GET /api/etiqueta_qrcode.php?acao=listar_etiquetas&os_id=123
   └─ Retorna: Array de etiquetas com QR-codes

POST /api/etiqueta_qrcode.php?acao=registrar_impressao
   └─ Parâmetros: etiqueta_id
   └─ Incrementa contador de impressões

GET /api/etiqueta_qrcode.php?acao=stats_impressoes
   └─ Retorna: Estatísticas por tipo

📐 FORMATOS DE NÚMERO
═════════════════════════════════════════════════════════════════════════════

Ordem de Produção:
   • Padrão: OS-001 (igual ao número da O.S.)
   • Múltiplos itens: OS-001-01, OS-001-02, etc
   • QR-code: OP|OS-001|123

Etiqueta O.S.:
   • QR-code: OS|OS-001|123
   • Contém: tipo|número|id

Etiqueta O.P.:
   • QR-code: OP|OS-001-01|123
   • Contém: tipo|número_op|id_os

🚀 COMO USAR
═════════════════════════════════════════════════════════════════════════════

1. ACESSAR GERADOR DE ETIQUETAS
   http://localhost/modules/os/gerar_etiquetas.php

2. GERAR ETIQUETA PARA O.S.
   ├─ Ir à aba "Ordens de Serviço"
   ├─ Clicar em uma O.S. da lista
   ├─ Preview do QR-code aparece
   ├─ Clicar "Imprimir" e configurar impressora
   └─ Etiqueta pronta para leitura de scanner

3. CRIAR ORDEM DE PRODUÇÃO
   ├─ Acessar /modules/os/ordem_producao.php
   ├─ Clicar "Nova O.P."
   ├─ Informar número da O.S.
   ├─ Sistema cria O.P. automaticamente
   ├─ Monitorar progresso dos itens
   └─ Marcar etapas como concluídas

4. INTEGRAÇÃO COM ESTOQUE
   ├─ QR-code gerado automaticamente
   ├─ Scanner lê na cada etapa
   ├─ Movimento registrado no banco
   ├─ Histórico disponível em tempo real
   └─ Rastreabilidade 100%

🎓 EXEMPLO DE USO VIA JAVASCRIPT
═════════════════════════════════════════════════════════════════════════════

// Gerar QR-code para O.S.
fetch('/api/etiqueta_qrcode.php', {
    method: 'POST',
    body: new FormData(document.querySelector('form'))
})
.then(r => r.json())
.then(data => {
    console.log('Etiqueta ID:', data.etiqueta_id);
    document.getElementById('qr').src = data.qr_url;
});

// Listar etiquetas de uma O.S.
fetch('/api/etiqueta_qrcode.php?acao=listar_etiquetas&os_id=123')
    .then(r => r.json())
    .then(data => console.log(data.etiquetas));
