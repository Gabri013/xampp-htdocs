📖 COMO USAR
═══════════════════════════════════════════════════════════════════════════════

PASSO 1: ACESSAR O GERADOR DE ETIQUETAS
────────────────────────────────────────────────────────────────────────────
URL: http://localhost/modules/os/gerar_etiquetas.php

Funcionalidades:
  • Aba "Ordens de Serviço" - Gerar QR-code para O.S.
  • Aba "Ordens de Produção" - Gerar QR-code para O.P.
  • Aba "Histórico" - Ver estatísticas de impressão

PASSO 2: SELECIONAR E GERAR ETIQUETA
────────────────────────────────────────────────────────────────────────────
  1. Selecione uma O.S. da lista
  2. Preview do QR-code aparece automaticamente
  3. Clique em "Imprimir" para abrir diálogo
  4. Configure impressora e formato
  5. Pronto! Etiqueta impressa

PASSO 3: CRIAR ORDEM DE PRODUÇÃO
────────────────────────────────────────────────────────────────────────────
URL: http://localhost/modules/os/ordem_producao.php

  1. Clique em "Nova O.P."
  2. Informe número da O.S.
  3. Sistema cria O.P. automaticamente
  4. Monitore progresso dos itens
  5. Marque etapas como concluídas

PASSO 4: ACOMPANHAR EM TEMPO REAL
────────────────────────────────────────────────────────────────────────────
  • Status atualiza em tempo real
  • Barra de progresso visual
  • Histórico de impressões disponível
  • Relatórios de estatísticas


🔌 API ENDPOINTS
═══════════════════════════════════════════════════════════════════════════════

POST /api/etiqueta_qrcode.php?acao=gerar_qr_svg
  └─ Parâmetros: os_id
  └─ Retorna: QR-code URL + etiqueta_id

POST /api/etiqueta_qrcode.php?acao=gerar_qr_svg_op
  └─ Parâmetros: op_numero, os_id
  └─ Retorna: QR-code para O.P.

POST /api/etiqueta_qrcode.php?acao=gerar_codigo128
  └─ Parâmetros: texto, os_id (opcional)
  └─ Retorna: Código de barras 128

GET /api/etiqueta_qrcode.php?acao=listar_etiquetas&os_id=123
  └─ Retorna: Array de etiquetas com QR-codes

POST /api/etiqueta_qrcode.php?acao=registrar_impressao
  └─ Parâmetros: etiqueta_id
  └─ Incrementa contador de impressões

GET /api/etiqueta_qrcode.php?acao=stats_impressoes
  └─ Retorna: Estatísticas por tipo

POST /api/etiqueta_qrcode.php?acao=excluir_etiqueta
  └─ Parâmetros: etiqueta_id
  └─ Remove etiqueta do banco


🗂️ TABELAS DO BANCO
═══════════════════════════════════════════════════════════════════════════════

etiquetas_impressas
  • id (PK)
  • os_id (FK) - Ordem de Serviço
  • op_numero - Número da O.P.
  • tipo (ENUM: qr_os, qr_op, codigo128)
  • conteudo - Dados do QR/Barcode
  • dados_qr (JSON)
  • impressoes (INT) - Contador
  • usuario_id (FK)
  • data_criacao, data_ultima_impressao

ordens_producao
  • id (PK)
  • os_id (FK)
  • numero (UNIQUE) - OP-001, OP-001-01
  • status (ENUM: pendente, em_producao, concluida, parada)
  • responsavel_id (FK)
  • data_inicio, data_termino, prazo_original
  • observacoes (TEXT)
  • criado_em, atualizado_em

ordens_producao_itens
  • id (PK)
  • op_id (FK)
  • os_item_id (FK)
  • quantidade, quantidade_produzida
  • valor_unitario
  • status (ENUM: pendente, produzindo, concluido)
  • data_conclusao

ordens_producao_etapas
  • id (PK)
  • op_id (FK)
  • etapa (VARCHAR: corte, dobra, solda, etc)
  • status (ENUM: pendente, em_producao, concluido, parado)
  • usuario_id (FK)
  • data_inicio, data_conclusao
  • observacao, sequencia

═══════════════════════════════════════════════════════════════════════════════

        (via /api/etiqueta_qrcode.php)
                ↓
        Imprimir Etiqueta
        (10x15cm ou A4)
                ↓
        Criar Ordem de Produção
        (via /modules/os/ordem_producao.php)
                ↓
        Monitorar Etapas
        (corte → dobra → solda → acabamento → embalagem)
                ↓
        Registrar Conclusão
        (Marcar etapas como concluídas)
                ↓
        Histórico Disponível
        (Dashboard com estatísticas)
