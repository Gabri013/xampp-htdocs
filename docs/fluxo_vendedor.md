# Fluxo de Trabalho - Vendedor

## 1. Vendas e Orçamentos
- **Dashboard Vendedor**: Visualiza vendas, orçamentos e OS
- **Nova Venda**: Cria venda com produtos
- **Orçamentos**: Cria/edita orçamentos

## 2. Gestão de OS Vinculadas
- **OS de Venda**: Criadas automaticamente ao vincular produtos
- **Acompanhar Status**: 
  - `em_projeto` → Projetista trabalhando
  - `proposta` → Projeto enviado para aprovação
  - `em_revisao` → Projeto em revisão
  - `em_producao` → Produção iniciada
- **Detalhes da OS**: Visualiza itens, arquivos do projetista, status

## 3. Arquivos de Projeto
- **Visualizar**: PDF, DXF e fotos anexadas pelo projetista
- **Baixar**: Clique no ícone de arquivo para download

## 4. Comunicação com Projetista
- Verifique o status da OS no dashboard
- Acompanhe a coluna "Status" da OS
- Entre em contato se necessário

## Fluxo Resumido
```
Nova Venda → OS Criada → Projetista (em_projeto) → Proposta → Aprovação → Produção → Concluída
```
